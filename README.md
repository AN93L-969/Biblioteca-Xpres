<h2>Qué es "Biblioteca Xpres"??</h2>

<p>Este repositorio contiene un proyecto full java en el cuál se busca simular un programa que a trabes de la
API Gutendex, busque en primer lugar los primeros 10 libros más descargados, luego estadísticas de los
primero 32 libros y finalmente permitir al usuario buscar un libro por medio del título. El proyecto es con fines
educativos y para desarrollo y practica personal.</p>
